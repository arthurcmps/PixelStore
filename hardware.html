Hardware

PLACAS DE VÍDEO

É um componente do computador que gera e renderiza imagens para a tela. Ela é responsável pela qualidade visual e desempenho dos aplicativos, e é fundamental para quem usa o computador para jogos ou programas pesados.

PLACAS MÃE

A placa-mãe une todos os componentes do computador, como o processador, a memória RAM, o disco rígido e a placa gráfica, em uma rede de fios.p>

MEMÓRIA RAM

É um tipo de memória volátil que armazena temporariamente os arquivos em uso no computador.

HDD/SSD

São dispositivos de armazenamento de dados em computadores, mas com diferenças significativas na forma como armazenam e acessam os arquivos

FONTES

As fontes de alimentação possuem função principal de transformar a corrente elétrica da rede, que chega em uma voltagem alta, para uma voltagem menor, que é a indicada para o hardware do computador.

PROCESSADORES

O processador, também conhecido como CPU (Central Processing Unit), é o componente de hardware responsável por executar tarefas e programas em um dispositivo eletrônico.

PLACAS DE INTERFACE

É uma peça física de hardware que conecta um computador a uma rede ou a uma interface de som.

GABINETES

É o item em que instalam todos os outros componentes do computador.
